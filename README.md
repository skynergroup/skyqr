# QR Code Generator

Generate QR codes for WiFi, URLs, text, email, SMS, contacts, and locations.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```

## Features

- WiFi credentials
- Website URLs
- Plain text
- Email addresses
- SMS messages
- Contact cards
- GPS locations
- Download as PNG
